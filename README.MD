# NestDeck

NestDeck is an open-source project that transforms a Google Nest Hub into a customizable touch dashboard for controlling applications, services, and automations.

The project aims to provide a modern, self-hosted alternative to commercial stream deck solutions through a web-based interface, a visual editor, and a modular backend.

## Goals

* Build a responsive dashboard optimized for the Google Nest Hub.
* Provide a visual editor for creating and organizing pages.
* Support integrations such as OBS Studio, Spotify, and desktop automation.
* Deploy everything with Docker.
* Keep the project modular and easy to extend.

## Status

NestDeck is currently in the early stages of development.

The initial focus is on building the project's architecture and core functionality before implementing integrations.
